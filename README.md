# Industrial-IOT-maintenance-system
## Project Description
This project is a Predictive Maintenance System using Machine Learning.
It helps to monitor machine health and predict failures in advance.

## Technologies Used
- Python
- Machine Learning (Random Forest)
- Pandas, NumPy
- Flask (if used)

## Files
- app.py → Main application
- model.py → Model training
- model.pkl → Saved model
- dataset.csv → Dataset

## How to Run
1. Install required libraries
2. Run app.py
3. View output in browser

## Output
Shows prediction of machine failure and maintenance alerts.
